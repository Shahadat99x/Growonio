## Confirmed Decisions

- Business name: Growonio
- Domain direction: growonio.com
- Primary market: Romania
- Language strategy: Romanian (default) + English (/en)
- Positioning: practical digital systems for service businesses
- Tone: premium, modern, clear, business-focused
- Main stack direction: Next.js + TypeScript + Tailwind CSS + Supabase + Cloudinary + Vercel
- i18n: next-intl
- Forms: React Hook Form + Zod
- UI components: shadcn/ui (selective)
- Email (contact form): Resend
- Auth (admin): Supabase Auth
- Analytics: Google Analytics 4 + Search Console
- Admin approach: hybrid CMS, not page-builder
- SEO strategy: Romanian-first local organic SEO from the beginning
